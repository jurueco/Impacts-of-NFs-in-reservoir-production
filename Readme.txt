1- This repository contains the inputs of the case studies of section 5.1-5.4 of the work presented by Rueda et al (2022):
  "Hydromechanical simulation of fracture propagation and reservoir production with multiscale fractures"

2- The inputs were executed using the multiphysic framework GeMA (Geo Modeling Analysis).

3- The output can be vizualized in the post-processors paraview or GID 

4 - More details about the installation of the framework for student use and user manual can be found at:
https://web.tecgraf.puc-rio.br/gema/gema/doxygen/html/index.html
